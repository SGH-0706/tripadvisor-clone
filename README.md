# Tripadvisor Clone – Travel Guide Website

A modern and responsive Tripadvisor-inspired travel guide website built using HTML and CSS. This project showcases a clean travel booking interface with destinations, hotels, restaurants, and activity sections designed using Flexbox, Grid, and responsive web design techniques.

## Features

- Responsive navigation bar with logo, search bar, and menu links
- Featured destinations and hotel listings
- Restaurant and activity recommendation sections
- Interactive hotel cards with images, descriptions, prices, and buttons
- Hover effects and modern UI styling using CSS
- Fully responsive design for mobile, tablet, and desktop screens
- Clean footer section 

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries

## Sections Included

- Header/Navbar
- Search Section
- Featured Destinations
- Popular Hotels
- Food & Restaurant Tours
- Footer

## Responsive Design

The website is optimized for:
- Desktop screens
- Tablets
- Mobile devices

- Creating reusable card components
- Styling with modern CSS techniques
